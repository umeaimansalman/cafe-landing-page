# Regard Café — Artisan Coffee House ☕

A modern, responsive café landing page designed to showcase a premium artisan coffee experience. This project focuses on elegant UI design, smooth user experience, and a visually rich presentation of café offerings.

##  Overview

**Regard Café** is a fictional artisan coffee brand built to demonstrate frontend development skills through a realistic, high-end café website. It includes carefully designed sections such as hero banner, menu showcase, customer reviews, and contact information.

##  Features

* Modern and responsive landing page design
* Elegant café branding and typography
* Dynamic menu section (coffee, pastries, desserts)
* Customer testimonials/reviews section
* Clean pricing layout for products
* Visually consistent UI/UX theme
* Mobile-friendly design


##  Design Style

The design follows a warm, minimal, and artisan aesthetic:

* Soft neutral tones
* Premium coffee imagery
* Clean spacing and modern layout
* Focus on readability and product presentation

## Tech Stack

* HTML5
* CSS3
* Responsive Design Principles

## 📸 Note

This is a **portfolio project** created for learning and demonstration purposes. All branding, pricing, and content are fictional.
